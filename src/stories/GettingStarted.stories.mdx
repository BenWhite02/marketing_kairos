# Getting Started with Kairos

This guide will help you get up and running with the Kairos Design System in your marketing application.

## üöÄ **Quick Start**

### Prerequisites

```bash
node >= 18.0.0
npm >= 8.0.0
react >= 18.0.0
typescript >= 4.9.0
```

### Installation

```bash
# Install the design system
npm install @kairos/design-system

# Install peer dependencies
npm install react react-dom @heroicons/react framer-motion clsx tailwind-merge

# Install development dependencies
npm install -D typescript @types/react @types/react-dom
```

### Basic Setup

1. **Import styles in your main CSS file:**

```css
/* src/index.css */
@import '@kairos/design-system/styles.css';
@import 'tailwindcss/base';
@import 'tailwindcss/components';
@import 'tailwindcss/utilities';
```

2. **Configure Tailwind CSS:**

```js
// tailwind.config.js
const kairosConfig = require('@kairos/design-system/tailwind.config');

module.exports = {
  ...kairosConfig,
  content: [
    './src/**/*.{js,jsx,ts,tsx}',
    './node_modules/@kairos/design-system/**/*.js'
  ]
};
```

3. **Set up providers:**

```tsx
// src/App.tsx
import { ThemeProvider, AccessibilityProvider } from '@kairos/design-system';

function App() {
  return (
    <ThemeProvider theme="light">
      <AccessibilityProvider>
        <YourApplication />
      </AccessibilityProvider>
    </ThemeProvider>
  );
}
```

## ‚ôø **Accessibility**

Kairos is built with accessibility in mind. All components include:

- **ARIA labels** and semantic markup
- **Keyboard navigation** support
- **Screen reader** optimization  
- **Focus management** and visual indicators
- **Color contrast** compliance (WCAG 2.1 AA)

### Accessibility Helpers

<Source code={`
import { 
  ScreenReaderOnly, 
  SkipTarget, 
  FocusRing,
  useAccessibility 
} from '@kairos/design-system';

function AccessibleComponent() {
  const { announce } = useAccessibility();
  
  return (
    <div>
      <ScreenReaderOnly>
        This content is only visible to screen readers
      </ScreenReaderOnly>
      
      <SkipTarget id="main-content" label="Main content">
        <main>
          <FocusRing color="blue">
            <button onClick={() => announce('Action completed')}>
              Accessible Button
            </button>
          </FocusRing>
        </main>
      </SkipTarget>
    </div>
  );
}
`} language="tsx" />

## üîß **Advanced Features**

### Form Handling

<Source code={`
import { useForm } from 'react-hook-form';
import { Input, Button, Card } from '@kairos/design-system';

function AdvancedForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();
  
  return (
    <Card>
      <form onSubmit={handleSubmit(onSubmit)}>
        <Input
          {...register('email', { required: 'Email is required' })}
          label="Email"
          type="email"
          error={errors.email?.message}
        />
        <Button type="submit" variant="primary">
          Submit
        </Button>
      </form>
    </Card>
  );
}
`} language="tsx" />
